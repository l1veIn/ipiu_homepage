# 安装pytorch

PyTorch（一个开源的机器学习库，用于计算机视觉和自然语言处理等应用）是一个相对直接的过程，可以通过几种不同的方法进行。以下是几种常见的安装方式： {{ className: 'lead' }}

## 使用 pip 安装

如果你已经知道你的系统配置和需要的 PyTorch 版本，你可以直接使用 Python 的包管理工具 pip 来安装 PyTorch。确保你的 pip 是最新版本，以避免安装过程中的兼容性问题。

打开终端或命令提示符，执行类似下面的命令：

```bash {{ title: 'linux' }}
pip install torch==1.10.0+cpu torchvision -f https://download.pytorch.org/whl/torch_stable.html
```

请根据实际情况替换版本号和其他参数。

## 使用 conda 安装

如果你使用的是 Anaconda 或 Miniconda，可以使用 conda 来安装 PyTorch。Conda 是一个开源的包管理系统和环境管理系统，它允许你创建隔离的环境来安装不同版本的软件包。

使用 conda 安装 PyTorch，可以在终端或命令提示符中执行如下命令：

```bash {{ title: 'Linux' }}
conda install pytorch torchvision torchaudio cudatoolkit=11.3 -c pytorch
```

这里的 cudatoolkit=11.3 表示安装 CUDA 工具包的版本。

## 注意事项

在安装 PyTorch 之前，请确保你的系统已经安装了 Python。PyTorch 支持多个版本的 Python，包括 3.6、3.7、3.8 和 3.9 等。
如果你的系统有 NVIDIA 的 GPU 并支持 CUDA，安装与 CUDA 兼容的 PyTorch 版本可以显著提高性能。确保选择的 CUDA 版本与你的显卡驱动兼容。
在某些情况下，你可能需要设置环境变量或者使用虚拟环境来避免与其他 Python 项目的依赖冲突。


通过以上任一方法，你应该能够成功安装 PyTorch，并开始你的机器学习项目。安装完成后，可以通过运行 Python 并尝试导入 PyTorch 来验证安装是否成功：

```bash {{ title: 'Python' }}
import torch
print(torch.__version__)
```
