# 常用数据集

在计算机视觉（Computer Vision，简称CV）领域，训练数据集是用于训练和评估机器学习模型的重要资源。以下是一些常用的CV领域训练数据集：

## ImageNet

<Row>
  <Col>
  <Properties>
  <Property name="描述">
    ImageNet是一个大规模的图像数据库，旨在用于视觉对象识别软件研究。它包含超过1400万张经过标注的图像，每张图像都标记有一个词或短语，这些标签通常对应于名词，如“狗”、“汽车”等。
  </Property>
   <Property name="用途">
    ImageNet广泛用于图像分类、目标检测、图像分割等任务。
  </Property>
   <Property name="规模">
    ImageNet的规模非常大，包含超过2万类别的图像，是举办ImageNet大规模视觉识别挑战赛（ILSVRC）的基础。
  </Property>
</Properties>

## COCO (Common Objects in Context)

<Row>
  <Col>
  <Properties>
  <Property name="描述">
    COCO数据集是一个大型的图像识别、分割和对象检测数据库。它包含了大量现实世界的图像，每张图像中都包含多个对象以及它们的标注。
  </Property>
   <Property name="用途">
    COCO数据集常用于对象检测、实例分割、语义分割、关键点检测等任务。
  </Property>
   <Property name="规模">
    COCO包含超过200,000张图像，每张图像有多个对象实例和5000个类别的标注。
  </Property>
</Properties>

  </Col>
</Row>


## PASCAL VOC

<Row>
  <Col>
  <Properties>
  <Property name="描述">
    PASCAL VOC（Visual Object Classes）是一个用于计算机视觉竞赛的数据集，它包含了20个类别的图像，每个类别都有大量的图像。
  </Property>
   <Property name="用途">
    PASCAL VOC主要用于图像分类和目标检测任务。
  </Property>
   <Property name="规模">
    PASCAL VOC包含约12,000张图像，每个类别大约有1,000张图像。
  </Property>
</Properties>
